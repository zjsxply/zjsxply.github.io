我目前是清华大学[深圳国际研究生院](https://www.sigs.tsinghua.edu.cn/)计算机技术专业二年级硕士研究生，导师是[郑海涛副教授](https://scholar.google.com/citations?user=7VPeORoAAAAJ)。
我本科毕业于[中国矿业大学](https://www.cumt.edu.cn/)数学与应用数学专业，受[邵虎教授](https://math.cumt.edu.cn/6d/ce/c9104a355790/page.htm)指导。

目前，我在百度担任 Agent 算法研究员（Summer Camp 人才计划），研究 [伐谋科研与自进化智能体](https://cloud.baidu.com/product/famou.html)，它是 [MLE-Bench](https://github.com/openai/mle-bench) 排行榜当前第一的 Agent。
此前，我曾在美团 [LongCat 基础模型团队](https://github.com/meituan-longcat) 和腾讯 [光子工作室群](https://guangzi.qq.com/) 担任研究实习生，分别从事代码 Agent 评测，以及用于仓库级代码生成的静态分析引导解码研究。
我以第一作者身份提出 [Natural-Language Agent Harnesses](https://arxiv.org/abs/2603.25723)，探索下一代智能 Agent 的新形态。

**研究兴趣：**

- **更具洞察力和创意能力的 Agent**：突破当前 AI 系统在高层设计与原创性发现上的瓶颈。
- **下一代智能 Agent 形态**：构建自然语言的 Agent Harness，让用户能够更容易地掌控 Agent 的行为和 token 消耗。
- **通用长时程 Agent**：构建面向真实生产力场景的通用长时程 Agent。

我的长期目标是打造能够高效产出有用工作，并被人类社会深度应用的 AGI Agent。
